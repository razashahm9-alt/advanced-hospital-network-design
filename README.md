# Advanced Hospital Network Design

## Project Overview

This project is an advanced hospital network infrastructure designed using Cisco Packet Tracer. The main objective of this project was to simulate a real-world hospital communication system with multiple departments connected through routers, switches, servers, and wireless networking.

The project demonstrates how different hospital sections can communicate securely and efficiently using structured network design and subnetting techniques.


## Departments Included

* IT Department
* Clinical Area
* General Ward
* Private Ward
* Reception & Billing Area
* Lobby Wireless Network


## Features Implemented

* Multi-router network topology
* Router-to-router serial communication
* IP addressing and subnetting
* DNS Server Configuration
* HTTP Server Configuration
* SMTP & FTP Server Setup
* Wireless guest/lobby network
* ICMP packet testing and simulation
* Centralized hospital communication system


## Technologies Used

* Cisco Packet Tracer

## Network Components

## Testing & Simulation

The network was tested using Packet Tracer simulation mode and ICMP packets to verify successful communication between multiple departments and devices.
